GitHub Project: LLM-Based YouTube Comment Analysis

Project #1: LLM-Based YouTube Comment Analysis and Moderation

Analyze and moderate YouTube comments using LLM classification and prompt engineering.

Overview

This project automates the analysis and moderation of YouTube video comments using a Large Language Model (LLM). It walks through:

  • Extracting comments from YouTube
  • Storing them in a SQLite database
  • Classifying sentiment, anger, spam, and whether a response is needed
  • Generating responses (manually or with the LLM)
  • Categorizing the comments
  • Visualizing and exporting the final dataset

Setup & Installation
  1. Clone or download this repository.
  2. Open the project in your preferred IDE (e.g. PyCharm).
  3. Create a virtual environment (optional but recommended).
  4. Install dependencies:
# Option A: If you have setup_requirements.py
python setup_requirements.py

# Option B: Directly from requirements.txt
pip install -r requirements.txt
Step-by-Step Pipeline
  1. Install Requirements: Use setup_requirements.py or pip install -r requirements.txt.
  2. Extract Comments: Use youtube-comment-downloader or existing JSON files in 01_YouTube or 02__json_split.
  3. Count JSON Lines: Run 02_count_rows.py to confirm the number of entries.
  4. Create a SQLite Database: Run 02_create_db.py & 03_load_json.py, then verify row counts.
  5. Prompt Classification (Sample): Test classification on ~25 comments.
  6. Predict Four Factors: Use db_moderate.py with OpenAI to classify sentiment, anger, spam, etc.
  7. Create Responses: 07_create_responses.py or 07_create_responses_llm.py for automated replies.
  8. Comment Categories: 08_categories.py to add a categories column or table.
  9. Analytics & Visualization: 09_visualization.py with Plotly or similar for charts.
  10. Export a Clean Dataset: 11_export.py to produce a clean_dataset.json.
  11. Data Pipeline: 12_pipeline.py automates everything end-to-end.
