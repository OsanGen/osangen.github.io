## 2) What this repo is
This is an Astro-based portfolio and learning-product site for OSAN. It hosts learn surfaces (Modules, Workshops, Labs), contributor flows (`/join`), proof/resume content, and GPT/game listings under a single OSAN v2 visual and motion framework.

### User-facing artifacts
- Public web routes driven by `src/pages/**` with shared shell in `src/layouts/BaseLayout.astro`.
- Learn split:
  - Workshops (`/workshops`)
  - Modules (`/visual-classes`) with one-at-a-time carousel flow
- Support surfaces: `/labs`, `/join`, `/gpts`, `/resume`, `/more`.
- Static build + dist validation (`npm run verify:dist`) and runtime smoke (`npm run verify:runtime`).

## 3) System map
- Core shell:
  - `src/layouts/BaseLayout.astro`: global metadata, nav/footer composition, motion mount, and route-level accent token.
  - `src/components/Header.astro` + `src/components/Footer.astro`: nav chrome; both consume `src/lib/site-nav.ts`.
  - `src/components/MotionEnhance.astro`: injects shared motion runtime host + top progress rail target.
  - `src/scripts/osan-motion.ts`: route-gated reveal/scroll behavior and reduced-motion safeguards.
  - `src/components/ServiceWorkerRegister.astro` + `public/sw.js`: caching/versioning/offline baseline.
- Data and catalogs:
  - `src/lib/content.ts` + `src/lib/schemas.ts`: content loading and typed validation.
  - `src/lib/workshop-catalog.ts`: workshop taxonomy + module filtering constants.
  - `src/lib/learn-pages.ts`: shared transformation helpers for modules and workshops.
  - `src/content/data/*` JSON content sources.
- Shared style system:
  - `src/styles/tokens.css`, `src/styles/global.css`, and route/page scoped stylesheet additions.
- Verification tooling:
  - `scripts/check-dist-integrity.mjs`, `scripts/runtime-http-smoke.mjs`
  - `npm` scripts for check/build/dist/runtime validation.
