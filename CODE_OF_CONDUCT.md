# Contributor Covenant Code of Conduct (adapted)

## Our Pledge

We pledge to make participation in this project open and welcoming.

## Our Standards

- Use respectful, clear language.
- Keep claims evidence-based.
- Use accessibility-minded design practices.
- Give constructive feedback and include next steps.

## Unacceptable Conduct

- Harassment, slurs, or personal attacks.
- Fake urgency, pressure tactics, or manipulative calls.
- Sharing private data without consent.

## Enforcement

Report concerns to the project maintainers. We review with fairness and focus on learning.

## Scope

This code applies to all project spaces: issues, merge requests, and community channels.
