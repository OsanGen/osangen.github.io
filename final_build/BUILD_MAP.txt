# BUILD_MAP
- `src/styles/tokens.css`: defines canonical OSAN/CogSec pastel design tokens and baseline surfaces.
- `src/styles/global.css`: global utility classes and component styles for layout, motion, nav, cards, and proof-first patterns using token palette.
- `src/components/Header.astro` + header styles in global.css: controls global shell navigation and Book CTA.
- `src/components/GPTCard.astro`: card row for each GPT directory entry (name, one-line outcome, tags, Open button).
- `src/pages/index.astro`: homepage router shell (hero + 3 route cards).
- `src/pages/workshops.astro`: workshop listing page with visual classes and workshop cards.
- `src/pages/workshops/[id].astro`: workshop detail page with proof line and teaching details.
- `scripts/generate-sitemap.mjs` + `public/sitemap.xml`: build-time sitemap generation.
- `src/lib/content.ts` + `src/lib/schemas.ts`: content loading, validation, and typing.
- `src/layouts/BaseLayout.astro`: layout shell and host page chrome, now includes service worker registration component.
- `src/components/ServiceWorkerRegister.astro`: registers the service worker in-browser for runtime caching/offline support.
- `public/sw.js`: offline/runtime caching service worker (`core`, `asset`, and `document` caches).
- `public/manifest.webmanifest`: manifest metadata for installability and offline entry behavior.
